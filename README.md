# 🧩 Organização do Projeto

## 📂 docs/

Responsável por armazenar todas as atividades desenvolvidas ao longo do semestre.

Inclui:

    Atividades organizadas por número
    Entregas finais
    Respostas e materiais complementares

## 📂 artefatos/

Núcleo técnico do projeto de qualidade de software.

Contém:

    🧠 diagramas/ → Modelagens (UML, fluxos, arquitetura)
    🧪 evidencias/ → Testes, prints e validações
    📑 relatorios/ → Documentação técnica detalhada
    📂 referencias/

Materiais acadêmicos, bibliografia e conteúdos de apoio utilizados.

📂 src/

Diretório reservado para implementação de código (quando aplicável ao projeto).

## 📦 Entregas

As entregas estão organizadas em:

    docs/atividade-XX/
    docs/entregas-finais/

Incluindo:

✔ Documentação
✔ Evidências
✔ Imagens
✔ Respostas
🧠 Metodologia

    🔍 Resolução de problemas reais

    📊 Documentação estruturada
    ⚙️ Aplicação prática da qualidade de software
    📊 Status do Projeto
